# Disney Pop-Up Book

This project started during a time when I was thinking a lot about my direction and future.
I chose quotes from Disney stories that felt meaningful to me, and translated them into a physical, interactive form.
Each page uses simple pop-up and sliding mechanisms to reveal the message, making the experience more personal and engaging.
Rather than focusing only on visuals, I wanted to explore how interaction can deepen the emotional impact of a story.
